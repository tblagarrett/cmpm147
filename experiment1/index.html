Experiment 1 - Getting Started

Description

Create a unique generator on glitch.com and move the working generator to my portfolio page. Work to understand how generators can work and the different languages involved in web development.

Technical

A grammar is defined with simple replacement tokens. A replacer() function picks a random replacement for any token that appears in the text. My personal contribution was solely by changing the different words that can be shuffled in, to make it a self-help book title generator

Reflection

I did not feel particularly motivated to work on this project, and so I simply did what was necessary to get it done. Perhaps I may return to work on it further, but for now this is where it will leave off.
