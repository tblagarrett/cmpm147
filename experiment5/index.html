Description

The goal of this experiment is to make a collection of shapes resemble a given image. This is done by placing them randomly and assessing how successful it was, iterating on the most successful versions and making it more precist by tuning down the mutation rate. As time goes by, a shape of the original image should become visible, though not perfect as this is an impression.

Technical

Carter: The picture is painted over by randomly generated circles. The way that the circles are generated is with the ellipse function and a mutation algorithm. The algorithm takes in hyperparams that slightly change how the circles behave when they are generated. The circles were able to be generated using the ellipse function because when you feed the function the same number for its diameter, it creates a circle. Garrett: Each triangle is made from a preset template, and are adjusted solely through position and rotation. The triangle size will scale with the size of the image, so that it can work with any size image. In total, there are 500 triangles throughout the image. As time progresses and the mutation rate narrows it becomes more and more precise, lending to long generation times and a slow process. James: I chose two manga pannels because they are black and white and they would have suited the line structure. Unforutnately, I only managed to create horizontal lines, so it didn't turn out how I hoped. I randomly generated 1500 rectangles with random positioning and length, but constrained the height to less than 7 pixels. It continually places the lines, on the canvas and mutates them trying to match the shapes of the manga pannels. It really doesn't come out nicely, since the lines are all horizontal and the pannels include lines in all directions.
