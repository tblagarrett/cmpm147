Experiment 3 - Alternate Worlds

Description

Dungeon:

For this first part, the goal was to understand how tilemaps work, mostly through hardcoding in small details. A lot of time was spent here figuring out how the autotiling works. I got a lot of bug fixing experience in, and it built up a good foundation for the second part

Technical

Dungeon:

This is made possible by placing random rectangles and making sure they do not collide. The rectangles are connected by the center if possible, and a chest is placed in a random one of the "rooms." All of it is done without noise, which plans to be changed in the next half
